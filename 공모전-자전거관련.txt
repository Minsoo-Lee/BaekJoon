1. 핵심 컨셉
- 자전거 동호회 사람들을 위한 자전거 여행 플래닝 앱 서비스

2. 사용 API
- 두루누비
- 카카오내비 (속도, 위치정보 등)
- 카카오로그인
- 카카오 공유하기(모임 만들 떄 초대하기 기능)

3. 기능
- 기본 기능
    i. 카카오로그인으로 가입
    ii. 초기화면(랜딩페이지)을 네이버 밴드 형식으로 보여주기
        만약 가입한 모임이 없다면 만들기 or 초대링크 받기
- 모임 만들기 화면
    i. 모임 이름 설정
    ii. 출발지 입력 => 차후에 코스 검색 탭에서 이 출발지를 기준으로 솔팅
    iii. 음.. 나머지는 다같이 얘기해보면 좋을듯??
- 게시판 탭
    i. 방장만 글 작성이 가능한 게시판
    ii. 공지사항, 사진첩 등으로 활용 가능
- 일정 탭
    i. 캘린더 형식이 아닌 게시글 형식
        ex) 5/18 한강투어
            5/23 나주 영산강 투어
            ...
    ii. 참 / 불참 토글 버튼으로 참여인원 확인
    iii. 게시판 탭을 기본 화면으로 두고, 일정은 모달 형식으로 띄워줌
    인프런 강의 커뮤니티 탭 참고 (사진은 썸네일이나 크게 띄워도 무방)
- 자전거 내비게이션 탭
    i. 카카오 맵 API 내의 자전거내비게이션 사용 => 네이티브에서 호출하기 때문에 앱 개발자 1명 필요할 것 같음
- 코스 검색 탭 : 이 부분을 강화해서 메인으로 쓸 수 있도록?? 혹은 메인화면으로 띄워준다던가
    i. 두루누비 API에서 받아온 코스 목록을 띄워줌
    ii. 맘에 드는 코스가 있으면 링크 복사 후 공유 or '공유하기' 버튼 자체로 공유
        피드를 만든다거나 하면 컨셉에 어긋날 수 있기 때문에 단순히 공유만 하는 거로
    iii. 혹은 찜하기와 같은 버튼으로 구성원도 볼 수 있게 해주는 기능
