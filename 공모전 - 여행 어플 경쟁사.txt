1. 여행 노트 : 코스대로 기록하는 여행 어플
 - 해당 어플의 특징
   i. 매일 업데이트 되는 여행 정보 => 유저들의 피드
   ii. 피드처럼 여행 기록
   iii. 예매 서비스(숙박, 티켓 등)
   iv. 당근페이와 같은 결제 서비스
 - 우리 어플의 차별성
   i. 피드처럼 여행을 기록하는 건 비슷
   ii. 하지만 저 어플은 피드를 '기록'하는 데에 초점을 두는 반면, 우리 어플은 여행을 '계획'하는 데에 초점을 두고 있음
   iii. 우리만의 UI 혹은 기능으로 타 유저들의 피드를 이용하여 여행 계획을 기록하는게 더 수월함
  
2. 여다 - 1분 여행일정
- 특징
   i. 여행 코스를 자동으로 추천
   ii. 유저의 여행 취향 기반
   iii. '난이도 최상 - 아이 동행 일정도 손쉽게'
- 차별성
   i. 여행 코스를 자동으로 추천하는 것이 아니라, 스스로 여행 계획을 짜고 싶은 사람들을 위한 어플
   ii. 해당 어플에서 보여주는 사진(미리 저장되어 있는 혹은 업데이트되는)에 국한되지 않고 타 유저들이 직접 찍은 사진들도 가능
   iii. 내가 다닌 여행지 관련 사진들도 피드로 보관 가능 

3. 위시빈 - 여행필수앱/자유여행 끝판왕(개인적으로 이 어플이 가장 비슷한 어플이 아닐까..?)
- 특징
   i. 블로그처럼 여행 일지를 기록한 글을 통해 인세 / 원고료를 받을 수 있음
   ii. 가고 싶은 곳 / 가본 곳을 나누어서 기록이 가능함
   iii. 호텔 / 항공권 등 예매 서비스
   iv. 자유 여행 일정을 만들 수 있음
- 차별성
   i. 아예 컨셉 자체를 '기록'과 '계획'에 초점을 맞춤
   ii. 해당 어플은 (내가 봤을 때) '기록'에 더 초점이 맞추어져 있음
      => 계획 기능쪽은 뭔가 부실한 느낌이 든다.
   iii. 우리 어플의 핵심 컨셉인 '기록'을 통한 '계획'으로 차별성을 둘 수 있지 않을까 싶음
   iv. 해당 어플은 내 글을 사람들이 많이 읽어야 이득이 있지만, 애초에 우리 어플은 그러한 이득을 생각하고 사용하지 않기 때문에 가벼운 마음으로 자유롭게 기록할 수 있음
   v. 해당 어플은 타 유저의 피드에서 끌어다가 쓸 수 있는 기능이 없음

4. 좀 더 차별성을 가질 수 있는 부가 기능
(어제 회의 때 나왔던 내용)
i. 포켓스탑과 같은 포토존
ii. 내일로를 신청하면 역마다 찍어주는 도장처럼 무언가 징표?를 남기는 기능
(내가 생각한 내용)
iii. 해당 피드에 위치(GPS)좌표가 나왔으면 좋겠음(카카오맵, 네이버지도처럼)
iv. 숙소 관련해서(국문 관광정보 서비스 쓰면 됨) 숙소의 정보 등도 표시할 수 있도록
   => 말 그대로 '숙소 정보'만 따로 볼 수 있게끔?
v. 여행 계획에 당장 추가하는게 아니라도 찜 목록과 같이 일단 저장해두는 기능

5. 그 외 의문점들
i. 피드에 집중(글) or 사진에 집중 (사진에 집중이라고 나오긴 했지만 혹시 모르니)
